# Kamal Lochan Sahu — Portfolio

> "I build systems that think, predict, and act — without being told twice."

## Project Structure

```
kamal-portfolio/
├── frontend/     → Next.js 16 (React, Tailwind, Framer Motion)
└── backend/      → FastAPI (Gemini AI, GitHub API, JD Matcher)
```

## Run Frontend
```bash
cd frontend
npm run dev
# → http://localhost:3000
```

## Run Backend (Script 3 onwards)
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
# → http://localhost:8000
```
